How to set up your Foundation project
1 Install Node.js version 11
2 Install Git
3 Open Node.js command prompt (Run as administrator)
4 Type: 
npm install --production windows-build-tools
5 Type: 
npm install --global foundation-cli
6 Go to the folder where the project will be (e.g. Desktop) 
cd Desktop
7 In the folder type: 
foundation new
8 Select "website"
9 Type the name of the project
10 Select ZURB to use all the tools
11 Go to the project folder
12 Type 
npm start
How to push your Foundation project
1 Open Node.js command prompt (Run as administrator) and go to the project folder
2 Type: 
git init
3 Type: 
git remote add remote-name git-url
Make sure to replace remote-name and git-url
4 Type: 
npm run build
5 Look for .gitignore file in te project and remove the line dist from there
6 Type: 
git add -A
7 Type: 
git commit -m "comment"
8 Type: 
git subtree push --prefix dist remote-name gh-pages
