IMDb Scraper

This scraper extract information from the Born Today section in imdb.com. You will get a text file with names of actresses and actors born on the same day you execute the script. Also, you will get the link to their profile in IMDb. If you want to execute this script from Linux make sure you add the path to python in the first line of code.
